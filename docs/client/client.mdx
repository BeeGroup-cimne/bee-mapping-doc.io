In order to run the platform successfully, we need to configure the backend and client part. First of all we have to install all
the dependencies said in [section](../intro.mdx).

# Getting Started
## Install Node Dependencies
To install all the dependencies of the project, you have to run the following command:

```shell
    npm install
```

## Run Project

```shell
    npm start
```

## Configure environment variables
    REACT_APP_API_URL=
    REACT_APP_DEFAULT_ONTOLOGY_ID=
    WDS_SOCKET_PORT=0
